SECTION 03 — GOOGLE PRO REALITY CHECK

اکانت Google Pro دقیقاً به چه دردی می‌خورد؟

نکته‌ای که باید بدون توهم روشن شود: اشتراک مصرفی Google AI Pro شامل اعتبار API نیست. این اشتراک به «شما» سرویس می‌دهد، نه به «محصول شما». API جمینای از مسیر جدای Google AI Studio فعال و جداگانه صورتحساب می‌شود (یک سطح رایگانِ محدود برای توسعه دارد).

ابزار روزمره تحقیق/نوشتن بنیان‌گذاربلهبخشی از سرویس Concierge فاز ۰ را عملاً با همین می‌دهی.
فراخوانی مدل از داخل کد RPIMخیرنیازمند کلید API از AI Studio + فعال‌سازی صورتحساب جدا — ردیف ۳ چک‌لیست.

SECTION 05 — MODEL STRATEGY

استراتژی مدل‌ها: چه مدلی، برای چه کاری، از کجا

هیچ «یک مدل برای همه‌چیز» وجود ندارد؛ Model Gateway بر اساس نوع کار مسیریابی می‌کند. این جدول پیکربندی پیشنهادی روز اول است — نه حکم ابدی. انتخاب نهایی ردیف T2 فقط با سنجه فارسی خودت انجام می‌شود، نه ادعای وبلاگ‌ها.
