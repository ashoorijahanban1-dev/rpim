# ایوال ۵۰-پرامپتی فارسی — انتخاب MODEL_T2

طبق اساسنامه، `MODEL_T2` تا قبل از این ایوال خالی می‌ماند و گیت‌وی برای
تسک‌های T2 پاسخ 503 می‌دهد. این پوشه همان ایوال است.

- `prompts_fa.jsonl` — ۵۰ پرامپت فارسی در ۵ دسته (۱۰تایی): پست با لحن برند،
  قلاب/CTA، پایبندی به زمینه (RAG)، خلاصه/بازنویسی، قضاوت ایمنی/QA.
- `run_eval.py` — کاندیداها را **از مسیر آداپترهای خود گیت‌وی** صدا می‌زند
  (نه SDK مستقیم)، خروجی هر مدل را در `results/` ذخیره و جمع‌بندی می‌کند.
  داور اختیاری LLM نمرهٔ ۱ تا ۵ می‌دهد؛ بدون داور، ستون نمره خالی می‌ماند
  تا انسانی پر شود.

## اجرا (روی سروری که کلیدها در env دارد)

```bash
export EVAL_CANDIDATES="gemini:gemini-2.0-flash,deepseek:deepseek-chat"
export EVAL_JUDGE="anthropic:claude-sonnet-5"   # اختیاری ولی توصیه‌شده
uv run python scripts/eval/run_eval.py
```

کلیدها فقط در env سرور: `GEMINI_API_KEY` / `DEEPSEEK_API_KEY` /
`ANTHROPIC_API_KEY` — هرگز در ریپو یا چت (قانون ۴).

## قاعدهٔ تصمیم

1. `results/summary.json` را ببینید؛ رتبه‌بندی بر اساس میانگین نمره و بعد هزینه.
2. حداقل ۱۰ خروجی برتر و ۵ خروجی ضعیف هر کاندیدا را انسانی مرور کنید —
   مخصوصاً دسته‌های `rag_grounded` (ادعای بی‌منبع؟) و `qa_safety`.
3. اگر صدرنشین در `rag_grounded` ادعای بی‌منبع می‌سازد، رد است — نفر بعد.
4. برنده را در Coolify (لگ آمریکا) به‌عنوان `MODEL_T2` ست و redeploy کنید؛
   قیمت‌های واقعی را هم در `providers.PRICES` به‌روز کنید (ADR 0013).

خروجی‌های `results/` را کامیت **نکنید** — حاوی خروجی مدل و متادیتای اجراست؛
فقط `summary.json` نهایی را در صورت نیاز به ADR پیوست کنید.
